# 工具介绍

# encode.exe

将 PNG 或任意文件编码为 base64。

示例：

将 cat.png 编码为 base64 并写入 cat.txt。

```encode.exe cat.png cat.txt```

# decode.exe

将 base64 文件解码为二进制文件。

示例：

将 cat.txt 从 base64 解码为二进制并写入 cat.png。

```decode.exe cat.txt cat.png```

# wav.exe

录制音频为 .wav 文件，例如作为 OMINI 模型的输入。

对录音进行放大处理。

示例：

输入命令后，长按 [SPACE] 键录音。

```wav.exe test.wav```
