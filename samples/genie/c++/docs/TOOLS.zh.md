# 工具介绍

# encode.exe

它可以帮助你将 png 或任何文件编码为 base64 文件。

示例：

此示例将 cat.png 编码为 base64 格式的数据并写入 cat.txt。

```encode.exe cat.png cat.txt```

# decode.exe

它可以帮助你从 base64 文件解码为二进制文件。

示例：

此示例将 base64 格式的 cat.txt 解码为二进制并写入 cat.png。

```decode.exe cat.txt cat.png```

# wav.exe

你可能需要向 OMINI 模型输入 .wav 格式的音频文件，它可以帮助你录制你的声音！

我们使用了一些算法和技术来放大你的声音。

示例：

输入命令后，长按 [SPACE] 键进行录音……

```wav.exe test.wav```
